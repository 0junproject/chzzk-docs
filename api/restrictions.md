# Restriction

활동 제한 API로 활동 제한 추가, 삭제, 목록 조회를 할 수 있습니다.\
활동 제한을 호출하려면 사용자 계정으로 인증하여 얻은 Access Token이 필요합니다.\
API Scope는 `활동 제한 추가`, `활동 제한 삭제`, `활동 제한 목록 조회`입니다.

***
